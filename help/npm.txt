======NPM镜像使用帮助======

总述
--------

中科大镜像站暂未提供 npm 镜像。

=====使用说明=====
编辑~/.npmrc，添加

registry=http://npm.mirrors.ustc.edu.cn/registry

若不想将本源设置为默认源，可在安装包时采用以下用法：

npm %%--%%registry http://npm.mirrors.ustc.edu.cn/registry install <packagename>

=====注意事项=====

不支持publish，若出现错误，请将.npmrc中的用户名密码部分注释掉，并删除缓存(rm -rf ~/.npm)重试
