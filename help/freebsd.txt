=====FreeBSD 镜像使用方法=====

<note important>
FreeBSD 项目[[https://www.freebsd.org/doc/en/articles/hubs/mirror-official.html#mirror-official-become|暂时不再接受新站点成为官方镜像源]]，且不向第三方镜像站点提供pkgng  rsync同步接口。因此，我们无法及时可靠地从官方站点同步数据，并暂时使用不包含pkgng的非官方站点作为同步上游。
鉴于以上原因，我们不建议您使用科大镜像站提供的内容。建议您使用官方支持的镜像源。您可以在 
https://www.freebsd.org/doc/handbook/eresources-web.html 找到官方镜像站的列表。

我们并非 FreeBSD 的官方源，所同步的文件受到非官方上游的限制。使用请自行承担风险。

受上游所提供文件限制，本站无法提供 pkgng 软件源。
</note>

====使用说明====
<note important>
以下使用说明已过期，请自行鉴别。

最新版的 FreeBSD 使用 pkgng 管理软件包，请查阅相关资料了解详情。
</note>

===Packages使用方法===

以FreeBSD 8.1 i386为例, 在终端下执行以下命令(操作前请做好相应备份)
<code>
setenv PACKAGESITE http://mirrors.ustc.edu.cn/freebsd/releases/i386/8.1-RELEASE/packages/Latest/
pkg_add -r package #package为要安装的软件名
</code>
===Ports使用方法===
以安装firefox为例, 编辑/etc/make.conf文件, 修改MASTER_SITE_BACKUP字段(操作前请做好相应备份)
<code>
MASTER_SITE_BACKUP?=http://mirrors.ustc.edu.cn/freebsd/distfiles/${DIST_SUBDIR}/
MASTER_SITE_OVERRIDE?=${MASTER_SITE_BACKUP}
</code>
然后在/usr/ports/www/firefox目录下执行以下命令(要先安装ports)
<code>
make
make install
</code>
